# python-zero-to-hero-discovery-driven
Structured 8-week Python to Full-Stack learning path with hands-on modules, self-tests, and portfolio-ready exercises.


This calendar defines the **daily learning rhythm** for the Discovery‑Driven Python → Full‑Stack course.

It is designed for adults (18+) with **60–90 minutes per day** and emphasizes progress through **guided discovery**, not lectures.

---

## Daily Learning Rules (Used Every Day)

1. Begin with a prepared runnable sandbox.
2. Start from a concrete real‑world analogy.
3. Follow the three‑period cycle:
   - Observe
   - Explore
   - Create
4. End with silent‑teacher self‑tests.
5. Publish one visible artifact every week.

---

## Daily Time Budget (60–90 Minutes)

- 10 min — Observe sandbox behavior
- 20–30 min — Guided Explore edits
- 25–35 min — Create a variation
- 10–15 min — Run tests + reflection notes
